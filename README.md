## Installation

1. Clone the repo
```bash
  git clone https://github.com/aryanGupta-09/React_Portfolio.git
```

2. Go to the project directory
```bash
  cd React_Portfolio
```

3. Install NPM packages
```bash
  npm install
```

4. Start the server
```bash
  npm start
```
